Before Infinite Options starts working on your store, you will need to install the app onto your current theme.

Did you install your store's theme before April 5th, 2018?

Follow the step-by-step instructions below, or watch the video for information on how to install Infinite Options.

 

Navigate to your product.liquid file.

Follow the instructions below to navigate to a file.

1. Starting from your Shopify dashboard, click on  Online Store, and then click on Themes.

2. Click the  Actions icon, then select Edit code. 

3. Use search bar on top left of the code editor to search for a desired file.

Do a search (ctrl F or ⌘ F) for the word  <form. Were you able to find it?

1. Once you find that line of code, click on the <form to highlight the ending </form tag. This designates the section of install code snippet (available in the next step) should be placed within (for example, in the image below, the section is located between lines 74 and 101).

2. Copy the Infinite Options install code snippet below. 

                      <div id="infiniteoptions-container"></div>

rd quantity< >. Paste the install code snippet above that, as seen below.

Not seeing a quantity section? Look for add to cart text and paste the install code snippet above that section, as seen below.

5. Save your changes.

6. You can now start creating options for your products. Follow the instructions in  this article to get started.
