1. Start by copying the code snippet below. 

    <div id="infiniteoptions-container"></div>

2. Navigate to your product.liquid file (Instructions on how to locate this file can be found here).

3. Do a search (ctrl F or ⌘ F) for the word  <form. Once you find that line of code, click on it to highlight the ending </form> tag. This designates the section of code that the copied snippet should be placed within (for example, in the image below, the section is located between lines 74 and 101).

Can't find <form in your product.liquid file? Follow the instructions in this article to locate it.

4.  Within this section of code, look for the word quantity. Paste the install code snippet above that, as seen below.

Not seeing a quantity section? Look for add to cart text and paste the install code snippet above that section, as seen below.

5. Save your changes.

6. You can now start creating options for your products. Follow the instructions in  this article to get started.

Updated Instructions (After April 5, 2018)

1. Start by copying the code snippet below. 

    <div id="infiniteoptions-container"></div>

2. Navigate to your product.liquid file (Instructions on how to locate this file can be found here).

3. Do a search (ctrl F or ⌘ F) for the word   {% form 'product'. Once you find that line of code, please locate the following {% endform %} tag a few code blocks below. This designates the section of code that the copied snippet should be placed within (for example, in the image below, the section is located between lines 135 and 183).

Can't find  {% form 'product' in your product.liquid file? Follow the instructions in this article to locate it.

4.  Within this section of code, look for the word quantity. Paste the install code snippet above that, as seen below.

Not seeing a  quantity section? Look for add to cart text and paste the install code snippet above that section, as seen below.

5. Save your changes .

6. You can now start creating options for your products. Follow the instructions in  this article to get started.
